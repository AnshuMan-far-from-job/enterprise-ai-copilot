# 📌 About the Project

Enterprise AI Copilot is a production-oriented AI assistant being built to explore modern LLM application architecture.

Instead of focusing only on AI model integration, this project emphasizes scalable backend engineering, modular design, conversation memory, prompt engineering, and enterprise-ready software architecture.

---

# ✨ Current Features

* ✅ FastAPI Backend
* ✅ Google Gemini Integration
* ✅ Modular LLM Provider Architecture
* ✅ Prompt Builder
* ✅ External System Prompt
* ✅ Conversation Memory
* ✅ REST API
* ✅ Clean Project Structure

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/anshumanrai041/enterprise-ai-copilot.git
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
python -m uvicorn app.main:app --reload
```

Open Swagger

```text
http://127.0.0.1:8000/docs
```
